This portfolio is a small static site built around a simple idea: the work and
the words should be the interface.

## Approach

The visual system uses one typeface, a restrained monochrome palette, and a
narrow reading column. The home page stays within a single viewport, while
project pages have room for longer case studies when they need it.

## Build

The site is generated with Astro and published as static HTML, CSS, and a tiny
amount of JavaScript. Projects live in Markdown files so they can be added or
edited without changing the site layout.
